# Technology Stack

## Backend
*   **Language:** Java 17+ (using modern features like Records and Sealed Classes)
*   **Core Framework:** Spring Boot 3.2.1
*   **Security:** Spring Security 6 with JWT (Stateless Authentication)
*   **Persistence:** Spring Data JPA
*   **Database:** PostgreSQL 14+
*   **API Documentation:** Knife4j (Swagger 3 / OpenAPI 3)
*   **Build Tool:** Gradle
*   **Utilities:** Lombok, Jakarta Bean Validation

## Infrastructure & DevOps
*   **Architecture:** Decoupled Frontend/Backend (RESTful API)
*   **Local Tunneling:** cpolar (for remote testing/demos)
